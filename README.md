🚀 HDPE Pipe Product Showcase Website

A modern, responsive product landing page built using HTML, CSS, and JavaScript with interactive UI components and smooth user experience.

📌 About the Project

This project is a fully designed product showcase webpage for HDPE pipes and industrial solutions.

It simulates a real-world company website with:

Product display
Technical specifications
Customer testimonials
FAQ section
Interactive UI components

The goal was to build a professional, production-level frontend UI using only core web technologies.

⚙️ Tech Stack
🧱 HTML5 – Structure
🎨 CSS3 – Styling & Layout (Responsive Design)
⚡ JavaScript (Vanilla JS) – Interactivity
